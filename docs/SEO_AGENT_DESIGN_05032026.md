### Модуль 2: Title/Meta оптимизатор (WF-203)
**Что делает:** берёт страницы с высокими показами но низким CTR →  
Claude генерирует новый title и description → отправляет задачу Саше

**Источники данных:**
- GSC: страницы с impression > 100, CTR < 5%, позиция < 15
- PostgreSQL: текущий title/description из БД продуктов

**Частота:** раз в неделю (понедельник)

**Выход:** Telegram сообщение:
```
📝 Title оптимизация · 3 страницы

1. /marketplace/standoff-2/currency
   Сейчас: "Standoff 2 Currency - Gaming Goods"
   Новый:  "Купить валюту Standoff 2 дёшево — мгновенно | GG Exchange"
   Ожидание: CTR 3.2% → 7%+ (+50 кликов/день)
   
[Саша: UPDATE products SET title=... WHERE slug=...]
```

### Модуль 3: Indexing Bot (WF-204) — ГЛАВНЫЙ
**Что делает:** ежедневно отправляет новые/обновлённые URL на индексацию

**Google Indexing API:**
- Квота: 200 URL/день
- Приоритет: новые продукты + страницы с обновлёнными ценами

**Яндекс Webmaster API:**
- Квота: 10 000 URL/день  
- Весь каталог за неделю

**Источник URL:** PostgreSQL — продукты с `updated_at > now() - interval '24h'`

```sql
SELECT slug, updated_at FROM products 
WHERE updated_at > now() - interval '24h'
ORDER BY updated_at DESC LIMIT 200
```

**Ожидаемый результат:** 18 000 "crawled not indexed" → < 5 000 за 3 месяца
