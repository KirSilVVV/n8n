# n8n on Render

Deployment of n8n workflow automation on Render.com with PostgreSQL.

## Deploy

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## Setup

1. Click "Deploy to Render"
2. Render will create:
   - Web Service (n8n) on port 5678
   - PostgreSQL database (free tier)
3. After deploy: open your service URL, register admin account
4. Go to **Settings → API → Create API Key**
5. Use that key for Claude Code MCP integration
